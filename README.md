
# Air Drawing with Fingertip Tracking

## Description
This is a simple **AI/ML + Computer Vision project** that lets you draw in the air using your **index finger**.  
It uses **Python**, **OpenCV**, and **MediaPipe** to track your hand and fingertip in real-time via webcam.

**Features:**
- Draw in real-time by moving your finger
- Mirror view (like looking in a mirror)
- Red line follows your fingertip
- Easy to extend with color change, erase, and save options
